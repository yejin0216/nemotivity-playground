<!-- GETTING STARTED -->
## Getting Started

### Prerequisites

* npm
```sh
npm install npm@latest -g
```

### Installation

1. Repository를 Clone합니다. 
```sh
git clone https://github.com/yejin0216/nemotivity-playground.git
```
2. NPM packages를 설치합니다. 
```sh
npm install
```
